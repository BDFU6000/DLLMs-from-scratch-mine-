# 02 · Machine Learning (Enhanced Edition)

Upgraded versions of the Machine Learning module with deeper beginner-friendly explanations,
embedded visuals, and real-world project examples.

## What's inside
| File | Topic |
|------|-------|
| `01_Types_of_Learning.md` | Supervised, unsupervised, semi-supervised, reinforcement learning |
| `02_Core_Algorithms.md` | Linear/logistic regression, trees, random forest, SVM, K-means, PCA |
| `03_Key_Concepts.md` | Overfitting, bias-variance, cross-validation, feature engineering, regularization |
| `images/` | SVG figures referenced by the markdown files |

## How the visuals work
- **Mermaid diagrams** are written directly inside the markdown and render automatically on GitHub.
- **SVG figures** live in `images/` and are referenced with relative paths, so keep the
  `images/` folder next to the `.md` files when you commit them.

Each file ends with a 🌍 real-world examples section and a 🧠 summary diagram.
