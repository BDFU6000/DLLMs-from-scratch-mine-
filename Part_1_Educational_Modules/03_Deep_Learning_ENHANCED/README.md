# 03 · Deep Learning (Enhanced Edition)

Upgraded versions of the Deep Learning module with deeper beginner-friendly explanations,
embedded visuals, and real-world examples.

## What's inside
| File | Topic |
|------|-------|
| `01_Neural_Network_Basics.md` | The artificial neuron, activation functions, network layers |
| `02_Loss_Optimizers_LR.md` | Loss functions, optimizers (SGD→AdamW), learning rate & schedules |
| `images/` | SVG figures referenced by the markdown files |

## How the visuals work
- **Mermaid diagrams** are written directly inside the markdown and render automatically on GitHub.
- **SVG figures** live in `images/` and are referenced with relative paths — keep the
  `images/` folder next to the `.md` files when you commit them.

Each file ends with a 🌍 real-world examples section and a 🧠 summary diagram, and includes
🔗 callouts linking concepts forward to LLMs.
