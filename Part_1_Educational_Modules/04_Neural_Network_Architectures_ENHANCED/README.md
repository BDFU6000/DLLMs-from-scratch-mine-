# 04 · Neural Network Architectures (Enhanced Edition)

Upgraded versions of the Architectures module with deeper beginner-friendly explanations,
embedded visuals, and real-world examples.

## What's inside
| File | Topic |
|------|-------|
| `01_FNN_and_CNN.md` | Feedforward nets (tabular) and Convolutional nets (images) |
| `02_RNN_LSTM_GRU.md` | Sequential data, memory, the vanishing-gradient problem, gates |
| `03_Transformers.md` | Self-attention, Q/K/V, multi-head, positional encoding, the Transformer block |
| `images/` | 11 SVG figures referenced by the markdown files |

## How the visuals work
- **Mermaid diagrams** (flowcharts, timeline, mindmaps) render automatically on GitHub.
- **SVG figures** live in `images/` and are referenced with relative paths — keep the
  `images/` folder next to the `.md` files when you commit them.

Each file ends with a 🌍 real-world examples section and a 🧠 summary, with 🔗 callouts linking
back to earlier modules and forward to LLMs. File 3 is the capstone: the architecture behind
every modern large language model.
