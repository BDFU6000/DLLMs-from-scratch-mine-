## How the visuals work
- **Mermaid diagrams** render automatically on GitHub.
- **Math** is written in LaTeX ($$...$$), which GitHub also renders.
- **SVG figures** live in `images/` and are referenced with relative paths — keep the
  `images/` folder next to the `.md` files when you commit them.

Each file ends with a 🌍 real-world section and a 🧠 summary, with 🔗 callouts linking back to
§2/§3/§4 and forward through the curriculum.
